## 使用

> 访问 http://localhost:8081/\_test/\_icon 查看图标样式

```
<gal-icon icon="iconName" size="iconSize"></gal-icon>
```

### props

| 属性名 | 说明     | 类型   | 必须  | 默认值 | 可选值     | 补充                                    |
| ------ | -------- | ------ | ----- | ------ | ---------- | --------------------------------------- |
| icon   | 图标类型 | String | true  |        | 见表格下方 |                                         |
| size   | 图标尺寸 | String | false |        |            | 当没有值时会设置元素当前的 font-size 值 |
