## 使用

```
<gal-icon-button
	icon="iconName"
	circle
	size="size"
></gal-icon-button>
```

### props

| 属性名 | 说明     | 类型   | 必须  | 默认值  | 可选值          | 补充                                                                  |
| ------ | -------- | ------ | ----- | ------- | --------------- | --------------------------------------------------------------------- |
| icon   | 图标类型 | String | true  |         | 见表格下方      | 同 [icon](./icon.md)                                                  |
| type   | 样式风格 | string | false | default | default primary | 在值为 default 设置透明背景，其它同 [gal-button](../button/button.md) |

-   其他 [gal-button](../button/button.md) 属性
