## 使用

```
<gal-icon-button
	icon="iconName"
	circle
	size="size"
></gal-icon-button>
```

### 属性

-   icon

    -   type: String
    -   required: true

    具体参见 [gal-icon](../icon/icon.md)

-   type

    同 [gal-button](../button/button.md)
    在值为 default 情况下添加 --button-bg-color: transparent; css 属性值

-   其他 [gal-button](../button/button.md) 属性
