## 使用

> 访问 http://localhost:8081/\_test/\_iconbutton 页面查看样式与使用

```
<gal-icon-button
	icon="iconName"
	circle
></gal-icon-button>
```

### props

| 属性名 | 说明     | 类型   | 必须 | 默认值 | 可选值     | 补充                 |
| ------ | -------- | ------ | ---- | ------ | ---------- | -------------------- |
| icon   | 图标类型 | String | true |        | 见表格下方 | 同 [icon](./icon.md) |
