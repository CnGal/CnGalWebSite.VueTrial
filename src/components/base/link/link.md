## 使用

```
<gal-link to="/">
......
</gal-link>
```

### props

| 属性名 | 说明     | 类型   | 必须  | 默认值 | 可选值 | 补充                                 |
| ------ | -------- | ------ | ----- | ------ | ------ | ------------------------------------ |
| to     | 跳转路径 | string | false | -      | -      | 会根据传入值的内容判断是否为外部链接 |
| href   | 跳转路径 | string | false | -      | -      | 外部链接                             |
