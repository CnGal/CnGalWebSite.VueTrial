## 使用

```
<gal-breadcrumb
    :item="
        [{
            value: '1',
            to: '/#'
        },
        {
            value: '2'
        }]
    "
></gal-breadcrumb>
```

### props

| 属性名    | 说明               | 类型   | 必须  | 默认值 | 可选值 | 补充                       |
| --------- | ------------------ | ------ | ----- | ------ | ------ | -------------------------- |
| item      | 文字内容和跳转路径 | Array  | true  | -      | -      | 会根据传入值的内容加载内容 |
| separator | 跳转路径           | string | false | /      |        | item 之间的分隔符          |
