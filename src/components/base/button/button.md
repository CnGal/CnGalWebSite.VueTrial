## 使用

```
<gal-button circle>
......
</gal-button>
```

### attrs

| 属性名 | 说明         | 补充 |
| ------ | ------------ | ---- |
| circle | 是否显示圆角 |      |

### props

| 属性名 | 说明     | 类型   | 必须  | 默认值  | 可选值          | 补充 |
| ------ | -------- | ------ | ----- | ------- | --------------- | ---- |
| type   | 样式风格 | string | false | default | default primary |      |
| width  | 显示宽度 | string | false | normal  | normal full     |      |
