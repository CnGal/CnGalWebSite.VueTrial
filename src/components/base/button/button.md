## 使用

```
<gal-button circle>
......
</gal-button>
```

### 属性

-   circle

    设置是否为圆形

-   width

    -   type: String
    -   default: "normal"

    设置 button 的显示宽度。normal 追随文字大小， full 100% 最大宽度

-   type

    -   type: String
    -   default: "default"

    设置 button 显示样式，当前支持值 default primary
