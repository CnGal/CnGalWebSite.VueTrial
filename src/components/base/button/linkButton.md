## 使用

```
<gal-link-button to="/" theme="solid">
......
</gal-link-button>
```

### props

| 属性名                               | 说明     | 类型   | 必须  | 默认值 | 可选值               | 补充 |
| ------------------------------------ | -------- | ------ | ----- | ------ | -------------------- | ---- |
| theme                                | 样式风格 | string | false | hollow | hollow solid outline |      |
| width                                | 显示宽度 | string | false | normal | normal full          |      |
| [gal-link](../link/link.md) 所有属性 |
