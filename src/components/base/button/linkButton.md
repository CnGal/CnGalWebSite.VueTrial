## 使用

> 访问 http://localhost:8081/\_test/\_linkbutton 页面查看样式与使用

```
<gal-link-button to="/">
......
</gal-link-button>
```

### props

| 属性名 | 说明     | 类型   | 必须  | 默认值 | 可选值             | 补充 |
| ------ | -------- | ------ | ----- | ------ | ------------------ | ---- |
| theme  | 样式风格 | string | false | text   | text solid outline |      |
| width  | 显示宽度 | string | false | normal | normal full        |      |
