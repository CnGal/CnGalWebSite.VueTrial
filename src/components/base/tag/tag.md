## 使用

```
<gal-tag :bgColor="string">
......
</gal-tag>
```

### attrs

| 属性名        | 说明           | 补充 |
| ------------- | -------------- | ---- |
| circle        | 是否显示圆角   |      |
| simple="true" | 显示为简约样式 |      |

### props

| 属性名  | 说明     | 类型   | 必须  | 默认值 | 可选值                                | 补充 |
| ------- | -------- | ------ | ----- | ------ | ------------------------------------- | ---- |
| bgColor | 背景颜色 | string | false | normal | normal primary success warning danger |      |
